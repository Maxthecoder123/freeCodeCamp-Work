Introduction

C++ is a cross-platform, object-oriented programming language that is an extension of the C language. It is widely used for developing applications that require performance and system-level access. C++ is known for its efficiency, flexibility, and strong focus on performance, making it popular for systems software, game development, and high-performance applications.

C++ contains a rich standard library that includes data structures like vectors, stacks, and maps, as well as utilities for handling time, input/output, and memory. In addition to the core language features, C++ can be extended with libraries and frameworks for specialized purposes. For example:

  • Client-side C++ can be used in combination with frameworks like Qt for developing desktop applications with graphical user interfaces (GUIs). This allows developers to build rich client applications that run efficiently on end-user machines.
  • Server-side C++ is often used in backend systems where performance is critical, such as handling high-traffic servers, databases, or real-time systems. C++ can communicate with databases, manage memory, and optimize performance in these environments.
What you should already know

This guide assumes you have the following basic background:

  • A general understanding of programming concepts like variables, control structures, and functions.
  • A good working knowledge of C, including pointers and memory management.
  • Some familiarity with object-oriented programming (OOP) concepts such as classes, inheritance, and polymorphism.
About C++

C++ and C share many similarities since C++ is derived from C, but they also have significant differences. C++ adds features such as object-oriented programming, which allows you to define classes and objects, and supports encapsulation, inheritance, and polymorphism. Unlike C, C++ provides features like function overloading, templates, and exception handling, making it a more versatile and expressive language.

In contrast to C's procedural programming style, C++ supports both procedural and object-oriented programming paradigms. C++ also introduces the Standard Template Library (STL), which provides powerful tools for working with collections of data like vectors, lists, and maps.

C++ offers fine-grained control over system resources, such as memory and hardware. However, developers need to manage memory explicitly (for example, using new and delete), which adds complexity and potential for errors if not handled properly.

Hello World

To get started with writing C++ code, open your favorite text editor or IDE and write your first "Hello world" program:

// Your First C++ Program #include int main() { std::cout << "Hello World!"; return 0; }

Compile and run this program in your terminal or IDE to see the output "Hello, World!" displayed on your screen.

Variables

In C++, you use variables as symbolic names for values in your program. The names of variables, called identifiers, follow certain rules.

A C++ identifier must start with a letter or underscore (_); subsequent characters can be letters, digits (0-9), or underscores. C++ is case-sensitive, so myVariable and MyVariable are considered two different identifiers.

Examples of legal variable names include numberOfHits, tempValue99, and _username.
